# Install these AND run the two setup commands below to automatically
# upgrade two components from their dependency-light fallback to a
# stronger ML-based version. The pipeline detects what's available at
# runtime — nothing else needs to change.

#   pip install sentence-transformers
#       -> semantic_similarity.py and vector_store.py switch from
#          TF-IDF cosine similarity to all-MiniLM-L6-v2 embeddings.
#
#   python -m spacy download en_core_web_sm
#       -> ner_extractor.py adds spaCy NER as a cross-check on top of
#          the heuristic name/organization extraction.
#

# downloading model weights from the internet (Hugging Face Hub /
# GitHub release assets respectively) on first use, which was blocked
# in the sandboxed environment this project was built in. They are
